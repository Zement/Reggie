feat(qpt): Phase 6 - Tile Picker Outlines and UI Improvements

Implement comprehensive tile picker enhancements with visual outlines, status indicators, and improved user interaction.

## Phase 6 Tile Picker Refinements

### Tile Picker Outlines & Visual Details
- Replace empty grid with minimalistic tile outlines using QPaint
- Add grass blade indicators with increased verticality on top edges
- Implement rounded corners on all corner tiles using QPainterPath
- Standardize corner tile designs using rotation-based templates
- Add stroke indicators to inner tiles for visual clarity
- Ensure outlines persist when tiles are assigned

### Click-to-Assign Functionality
- Implement mouse click handling on tile picker canvas
- Map grid coordinates to position types using CAT1_LAYOUT
- Remove position selector buttons (replaced by canvas clicks)
- Assign selected objects directly by clicking grid cells
- Provide visual feedback for tile assignments

### Status Indicator & Clear Button
- Add red/green status circle at top-left of canvas
  - Red: Indicates incomplete assignments
  - Green: Indicates all tiles assigned
- Add "CLEAR" button at bottom-right of canvas
- Clear button removes all assignments and resets canvas
- Status indicator updates automatically on tile assignment

### Tileset Slot Management
- Connect tileset selector to tile picker updates
- Automatically clear tile picker when switching tileset slots (Pa0-Pa3)
- Maintain brush state across tileset changes
- Prevent data inconsistency when changing tilesets

### UI/UX Improvements
- Status indicator positioned as widget overlay (not scene item)
- Clear button positioned as hovering overlay at bottom-right
- Proper repositioning on view resize
- Status indicator persists across canvas redraws
- Improved visual hierarchy and consistency
